## 비즈니스 로직의 가치를 높이는, 협업 중심의 프론트엔드 개발자 류정윤입니다.

**"복잡한 코드를 체계적으로 구조화하여 팀 전체의 생산성을 높이는 과정에 큰 보람을 느낍니다."**

단순히 기능을 구현하는 것에 그치지 않고, 동료들과 더 효율적으로 일하기 위해 무엇이 필요한지 고민합니다. 
레거시 페이지를 전수 조사해 문서화하거나 팀 내 Git-flow를 도입하는 등, 작은 체계를 세워 업무의 가시성을 확보할 때 큰 보람을 느낍니다. 
팀 전체가 비즈니스 로직에 더 집중할 수 있도록 돕는 든든한 프론트엔드 개발자가 되고자 합니다.

### ✨ 핵심 역량

- **구조적 안정성 확보:** TypeScript 기반의 타입 설계와 Git-flow 준수로 유지보수 가능한 코드 베이스 구축
- **DX(개발자 경험) 개선:** 파편화된 소스 자산화 및 환경 표준화를 통해 온보딩 비용 및 협업 마찰 최소화
- **데이터 기반 시각화:** MQTT와 Chart.js를 활용한 실시간 데이터 대시보드 구현 및 인터페이스 통합 경험
